# Contributing to Agent Factory

Agent Factory is an open platform. Contributions to the launchpad, agent tooling, and documentation are welcome.

---

## What We're Looking For

- **Launchpad improvements** — better deploy flows, new agent capabilities, UI/UX refinements
- **Agent features** — new automation types, scheduling, strategy modules
- **Documentation** — clearer explanations, guides, tutorials
- **Bug reports** — if something is broken, open an issue
- **Integrations** — new DEX support, analytics dashboards, notification systems

---

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Open a pull request with a clear description of what you changed and why

---

## Code Style

- JavaScript (CommonJS)
- Keep dependencies minimal
- Simple > clever

---

## Reporting Issues

Open a GitHub issue. Include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Environment (OS, Node version, browser)
