# Changelog

All notable changes to Agent Factory are documented here.

---

## [0.2.0] — 2026-03-18

### Added
- Agent dashboard with Phantom wallet integration
- Twitter/X API key management per agent
- Tweet posting directly from dashboard
- Buyback and sell trading actions from dashboard
- Agent card expand/collapse with status badges
- Canvas pixel rain background animation

### Changed
- Deploy wizard now includes image upload for agent profile pic
- IPFS metadata includes telegram and website fields

---

## [0.1.0] — 2026-03-15

### Added
- Initial release — Express server, deploy wizard, agent CRUD
- Agent creation wizard with multi-step deploy flow
- Token deployment via pump.fun (IPFS metadata upload + on-chain launch)
- Phantom wallet connect flow
- pump.fun token creation pipeline
- PumpPortal integration for buyback/sell transactions
- Retro pixel UI with custom styling
- JSON file database for agent persistence
- REST API for agents, deploy, trade, and tweet operations
